# Vue Twig Scaffold

This is a simple scaffold example built with Vue.js and the Twig template engine.

### Requirements

- PHP 7.1
- Node 8.4

### Install

```bash
composer install
npm install
```

### Build Tools

```bash
npm run gulp build
npm run gulp watch
```

### Start Server

```bash
npm start
```

### Testing

```bash
npm test
```
